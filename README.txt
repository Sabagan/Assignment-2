The Program Compiles on my Laptop

Aspects of the program that work:
- DOWNLOAD, DOWNLOADA, DOWNLOADG
- SEARCH, SEARCHA, SEARCHG, SEARCHP
- STORE, SONGS, BOOKS, PLAYLISTS
- PLAYSONG, PLAYBOOK, BOOKTOC
- ARTISTS, DELSONG
- MAKEPL, PRINTPL, ADDTOPL, DELFROMPL, PLAYALLPL, PLAYPL
- SORTBYLENGTH, SORTBYYEAR, SORTBYNAME

All the required parts of the program work and that
includes the bonus as well.

Some things to note:
- Associates of Podcast and Season have been removed
- There are extra exception thrown to handle extra cases
- PlayList does not allow duplicates
- Search and Download is case-sensitive
